# DobSandbox
The Source Code of DobSandbox Game
# Controls
<br>Arrows  - to move
<br>Escape - pause game
<br>F1 - debug menu 
<br>F3 - to enable or disable music 
<br>D - enable or disable delete mode 
<br>Click on blocks to select it

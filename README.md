**DobSandbox**
================

**The Source Code of DobSandbox Game**

**Controls**

* Arrows: Move
* Escape: Pause game
* F1: Debug menu
* F3: Enable/Disable music
* D: Enable/Disable delete mode
* Click on blocks to select it
* p - more visibility
* 0 - disable more visibility

# DobSandbox
The Source Code of DobSandbox Game
# Controls
Arrows  - to move 1
Escape - pause game 2
F1 - debug menu 3
F3 - to enable or disable music 4
D - enable or disable delete mode 5
Click on blocks to select it 6
